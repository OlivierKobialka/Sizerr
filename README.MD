# Sizerr - A simple Size Calculator

[Project for DataBase with SQL database integration and TypeScript React
](https://github.com/OlivierKobialka/Sizerr/blob/master/Description%20of%20Assumptions.txt)

## How to run

### Client Side

```bash
npm install

npm run dev
```

### Server Side

```bash
npm run start
```
